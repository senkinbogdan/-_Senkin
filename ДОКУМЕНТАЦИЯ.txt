1. ОПИСАНИЕ СИСТЕМЫ
================================================================================

Система учета рабочего времени Роснефти - это приложение для автоматического
отслеживания и управления рабочим временем сотрудников.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
• Регистрация и авторизация пользователей
• Автоматический учет времени работы в системе
• Визуализация рабочего времени по дням недели (тепловая карта)
• Ручное редактирование записей о рабочем времени
• Просмотр статистики и профиля пользователя
• Цветовая индикация рабочего времени

ОСОБЕННОСТИ:
• Корпоративный дизайн в цветах Роснефти (черный, желтый, белый)
• Логотип компании на всех страницах
• Современный интерфейс с вкладками
• Автоматическое сохранение данных

3. РЕГИСТРАЦИЯ НОВОГО ПОЛЬЗОВАТЕЛЯ
================================================================================

ШАГ 1: Открытие формы регистрации
• На странице входа нажмите желтую кнопку "Регистрация"

ШАГ 2: Заполнение данных
Обязательные поля (отмечены *):
• Имя *
• Фамилия *
• Отчество (необязательно)
• Email * - используется для входа, должен содержать "@"
• Телефон (необязательно)
• Пароль *
• Подтверждение пароля * - должно совпадать с паролем
• Должность *

ШАГ 3: Завершение регистрации
• Проверьте правильность введенных данных
• Нажмите черную кнопку "Зарегистрироваться"
• При успешной регистрации появится уведомление
• Вы автоматически войдете в систему

ВАЖНО:
• Email должен быть уникальным (один на пользователя)
• Пароль хранится в зашифрованном виде (SHA-256)
• После регистрации создается пустая таблица учета времени

ВОЗМОЖНЫЕ ОШИБКИ:
"Пользователь с такой почтой уже существует"
   → Используйте другой email адрес

"Пароли не совпадают"
   → Проверьте правильность ввода паролей

"Неправильный формат почты"
   → Email должен содержать символ "@"

6. РЕДАКТИРОВАНИЕ РАБОЧЕГО ВРЕМЕНИ
================================================================================

АКТИВАЦИЯ РЕЖИМА РЕДАКТИРОВАНИЯ:
1. Нажмите желтую кнопку "✏️ Редактировать"
2. Ячейки таблицы станут белыми
3. Появятся кнопки "💾 Сохранить" (зеленая) и "❌ Отмена" (красная)
4. Подсказка изменится на желтую

ИЗМЕНЕНИЕ ДАННЫХ:
Способ 1: Двойной клик
• Дважды кликните на ячейку нужного дня
• Появится курсор для редактирования
• Введите новое значение часов

Способ 2: Клавиатура
• Выберите ячейку (один клик)
• Начните вводить число
• Старое значение заменится на новое

ПРАВИЛА ВВОДА:
• Используйте точку (.) или запятую (,) для десятичных дробей
• Примеры корректных значений: 8, 8.5, 7.25, 10.75
• Диапазон: от 0 до 24 часов
• Можно вводить любые числа в этом диапазоне

ВАЛИДАЦИЯ ДАННЫХ:
При попытке сохранить система проверяет:
✓ Все значения являются числами
✓ Нет отрицательных значений
✓ Значения не превышают 24 часа

СОХРАНЕНИЕ ИЗМЕНЕНИЙ:
1. После редактирования нажмите "💾 Сохранить"
2. Система проверит корректность данных
3. При успехе появится уведомление "Изменения успешно сохранены!"
4. Таблица вернется в режим просмотра
5. Цвета ячеек обновятся согласно новым значениям

ОТМЕНА ИЗМЕНЕНИЙ:
1. Нажмите кнопку "❌ Отмена"
2. Появится запрос подтверждения
3. При подтверждении все изменения будут отменены
4. Восстановятся исходные значения

ОСОБЕННОСТИ РЕЖИМА РЕДАКТИРОВАНИЯ:
• Увеличенный размер ячеек (60px высота, 90px ширина)
• Крупный шрифт (14pt)
• Белый фон для удобства ввода
• Автообновление приостановлено
• После сохранения автообновление возобновляется

ВОЗМОЖНЫЕ ОШИБКИ:
"Количество часов не может быть отрицательным (Понедельник)"
   → Введите положительное число

"Количество часов не может превышать 24 в сутки (Вторник)"
   → Введите значение от 0 до 24

"Некорректное значение для дня Среда. Введите число (например: 8 или 8.5)"
   → Проверьте формат ввода (только цифры)

• PyQt6
• SQLite3

ФАЙЛЫ ПРИЛОЖЕНИЯ:
iub.py - главный файл приложения
user_tabs.py - модуль с функционалом вкладок
styles.py - файл стилей (CSS)
interface_1.ui - файл интерфейса Qt Designer
users_1.db - база данных пользователей (создается автоматически)
rosneft.png - логотип компании

БАЗА ДАННЫХ:
Таблица users:
• id - уникальный идентификатор
• name - имя
• surname - фамилия
• patronymic - отчество
• email - адрес электронной почты (уникальный)
• phone - телефон
• password - хэш пароля (SHA-256)
• position - должность

Таблица usage_hours:
• id - уникальный идентификатор
• user_id - связь с пользователем

11. УСТРАНЕНИЕ НЕПОЛАДОК
================================================================================

ПРОБЛЕМА: Приложение не запускается
РЕШЕНИЕ:
1. Проверьте установку Python и PyQt6
2. Установите зависимости: pip install PyQt6
3. Проверьте наличие всех файлов в папке проекта
4. Запустите из командной строки для просмотра ошибок

ПРОБЛЕМА: "Не удалось подключиться к базе данных"
РЕШЕНИЕ:
1. Проверьте наличие файла users_1.db
2. Проверьте права доступа к папке
3. Удалите users_1.db (будет создан заново, данные потеряются!)
4. Запустите приложение от имени администратора

ПРОБЛЕМА: Логотип не отображается
РЕШЕНИЕ:
1. Проверьте наличие файла rosneft.png в папке проекта
2. Убедитесь, что файл не поврежден
3. Проверьте название файла (регистр важен)

ПРОБЛЕМА: Время не сохраняется
РЕШЕНИЕ:
1. Дождитесь 10 секунд (автосохранение)
2. Используйте кнопку "Выход" вместо закрытия через диспетчер задач
3. Проверьте права записи в папку с базой данных

ПРОБЛЕМА: Не могу войти с правильным паролем
РЕШЕНИЕ:
1. Проверьте раскладку клавиатуры
2. Проверьте Caps Lock
3. Убедитесь, что email введен правильно (без лишних пробелов)
4. Попробуйте зарегистрироваться заново с новым email

ПРОБЛЕМА: Ошибка при редактировании времени
РЕШЕНИЕ:
1. Используйте только цифры и точку/запятую
2. Убедитесь, что значение от 0 до 24
3. Не используйте буквы или спецсимволы
4. Пример правильного ввода: 8.5

ПРОБЛЕМА: Окно слишком маленькое/большое
РЕШЕНИЕ:
1. Измените размер окна перетаскиванием границ
2. Минимальный размер: 400x500 пикселей
3. Рекомендуемый размер: 420x640 пикселей

ПРОБЛЕМА: Цвета отображаются неправильно
РЕШЕНИЕ:
1. Проверьте настройки цветопередачи монитора
2. Убедитесь, что используется тема Fusion
3. Перезапустите приложение
